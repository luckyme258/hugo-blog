Tri笔记，类似 Notion, 是一款笔记软件。
https://github.com/TriliumNext/Trilium

*   特点
    *   支持服务器同步
    *   支持个人用电脑作为服务器使用
    *   支持docker 搭建，简单快捷
    *   支持局域网实时同步数据
    *   只要电脑开机，打开doker服务，运行tri笔记，那么用任何设备都可以访问
    *   只要这些设备连接同一个无线网
    *   免费方案： 笔记本运行，使用这个笔记本的浏览器访问 https://www.local:8080
    *   支持同一个无线网下的任意设备，只需要访问网址，输入你设置的密码即可
*   部署方法
    *   笔者是Debian12, 你使用 Linux 任意一款都可以，只要运行Docker即可
    *   Docker 安装
        *   什么是docker? 你可以看作一个系统，可以运行许多软件

    *   Trilium Note 安装
    *   验证Docker 已经安装
    *   验证Trilium Note 已经安装
    *   启动Docker
    *   有用的命令
        *   docker ps 查看运行的容器都有谁
        *   docker ps -a 查看所有容器都有谁，包括离线的
    *   启动 Trilium
        *   docker start trilium 
        *   使用docker启动tri笔记
    *   设置tri笔记：tri笔记安装并且启动
        *   直接使用：可以在 浏览器 访问 https://www.local:8080
        *   其他设备使用
            *   设置内网穿透
                *   本机ip + localhost端口
                *   本机ip 通过 命令行查到
