## 磁盘空间清理指南

### 1. 查看磁盘占用
{{< code lang="bash" >}}
df -h
{{< /code >}}

### 2. 清理用户缓存
**查看缓存大小：**
{{< code lang="bash" >}}
du -sh ~/.cache
{{< /code >}}

**清理所有用户缓存（保留浏览器 Cookies）：**
{{< code lang="bash" >}}
rm -rf ~/.cache/*
{{< /code >}}

### 3. 清理系统日志
{{< code lang="bash" >}}
# 保留最近 100MB 的日志
sudo journalctl --vacuum-size=100M

# 保留最近 3 天的日志
sudo journalctl --vacuum-time=3d
{{< /code >}}
