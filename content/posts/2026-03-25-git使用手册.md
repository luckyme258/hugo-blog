###  打印状态
{{< code >}}
git status
{{< /code >}}

###  添加新文件
- #### 添加所有文件
{{< code >}}
git add .
{{< /code >}}

- #### 添加特定一个文件
{{< code >}}
git add content/posts/2026-03-05-notes.md
{{< /code >}}


- #### 添加多个符合格式的文件
{{< code >}}
git add content/posts/2026-03-25-*.md
{{< /code >}}


###  添加简单记录
{{< code >}}
git commit -m "内容"
{{< /code >}}


###  推送云端
{{< code >}}
git push
{{< /code >}}
