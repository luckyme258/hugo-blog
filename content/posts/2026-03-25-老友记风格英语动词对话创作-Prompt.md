核心要求（必须做到）
----------

1.  格式固定：**\- 场景：xxx，人物动作神态**\+ 换行**人物名：台词 + 适配 emoji**，无任何加粗 / 斜体 / 多余标点，纯文本呈现

2.  人物设定：严格使用老友记核心角色（Rachel/Monica/Phoebe/Chandler/Ross/Joey），每段对话 2 人且**角色不重复**，贴合角色性格（如 Phoebe 古灵精怪、Chandler 爱吐槽、Monica 强迫症、Joey 呆萌）

3.  台词要求：紧扣指定动词**动词含义**创作，台词中**动词至少出现 1 次**；全英文口语化，无难词 / 复杂句式，符合老友记日常唠嗑语气，带轻微吐槽 / 调侃 / 暖心感

4.  场景设定：均为老友记经典场景（中央公园咖啡馆 / 莫妮卡公寓 / 公寓楼下 / 阳台 / 中央公园草坪 / 湖边等），场景描述带简单人物动作神态，画面感强

5.  细节适配：台词后加**1 个贴合语境的基础 emoji**，无堆砌；单段对话台词简短，符合日常交流节奏，不冗长

6.  风格贴合：还原老友记「轻松吐槽 + 暖心搭茬 + 偶尔小调侃」的核心氛围，对话自然不生硬，像角色真实会说的话

避坑要求（必须避免）
----------

1.  避免角色重复：同一段对话不出现相同角色，整组创作中角色搭配尽量多样化，不一直固定某 2 人

2.  避免动词误用：不偏离动词核心含义，不将动词当名词 / 形容词使用，保证台词中动词用法准确

3.  避免复杂表达：无生僻词、长难句、语法复杂的从句，不出现书面化 / 正式化表达，全程口语化

4.  避免格式混乱：不修改固定格式，不增减符号，场景描述不与台词混行，人物名与台词间仅用冒号连接

5.  避免场景违和：不出现老友记中无相关的场景，不加入与角色性格不符的台词（如 Joey 不说复杂科普、Ross 不做无厘头的事）

6.  避免 emoji 滥用：每段台词仅加 1 个 emoji，不使用复杂 / 小众 emoji，emoji 必须贴合台词语境，不强行添加

7.  避免台词生硬：不刻意堆砌动词，台词自然融入动词，不出现「为了用动词而说的话」，符合日常交流逻辑

8.  避免多余内容：不添加角色介绍 / 台词解析 / 额外备注，仅保留「场景 + 人物台词」的核心内容

简洁执行版（直接复制使用）
-------------

按以下要求创作老友记风格英语短对话，**必须做到**：1. 固定格式「- 场景：xxx，人物动作神态 \\n 人物名：台词 + emoji」；2. 老友记核心角色 2 人（不重复），贴合性格；3. 紧扣指定动词含义，台词中动词至少出现 1 次；4. 经典场景，口语化无难词，带轻微吐槽 / 暖心感；5. 台词后加 1 个贴合语境 emoji。**必须避免**：角色重复、动词误用、复杂表达、格式混乱、场景违和、emoji 滥用、台词生硬、多余内容。单段对话为 2 人简短交流，还原老友记日常唠嗑氛围。
