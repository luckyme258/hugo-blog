# Hugo 代码块短代码使用说明

## 📦 安装方法

### 1. 创建短代码文件
将代码保存为 `layouts/shortcodes/code.html`

### 2. 目录结构
```
your-hugo-site/
├── layouts/
│   └── shortcodes/
│       └── code.html          # 将完整代码放入此文件
├── content/
│   └── posts/
│       └── example.md         # 使用短代码的 Markdown 文件
```

## 🚀 基本用法

### 最简单的用法

## 🎨 功能特性

### 1. 自动语法高亮
- 普通代码显示为蓝色（`#1a6d8f`）
- 注释显示为灰色（`#7f8c8d`）
- 支持行内语法高亮（可自定义）

### 2. 行号显示
- 自动为每一行添加行号
- 行号样式：灰色，右对齐

### 3. 复制功能
- **复制单行**：点击每行前面的 ❤️ 或 ⭐ 图标
- **复制全部**：点击右上角的"📋 复制全部"按钮
- 复制成功会显示提示消息

### 4. 视觉效果
- 毛玻璃背景效果
- 悬停动画
- 渐变边框
- 自定义滚动条

## 🎨 自定义颜色

### 修改主题色
在 CSS 的 `:root` 部分修改以下变量：

```css
:root {
  --code-blue: #3498db;        /* 主色调 */
  --code-dark-blue: #2980b9;   /* 深色主调 */
  --code-text: #1a6d8f;        /* 代码文字颜色 */
  --code-comment: #7f8c8d;     /* 注释颜色 */
  --code-line-number: #95a5a6; /* 行号颜色 */
}
```

### 修改代码文字颜色
如果想改变普通代码的颜色，修改 `--code-text` 变量：

```css
:root {
  --code-text: #0066cc;  /* 改为纯蓝色 */
  /* 或 */
  --code-text: #2c3e50;  /* 改为深灰色 */
  /* 或 */
  --code-text: #008080;  /* 改为青色 */
}
```

## ⚠️ 注意事项

1. **HTML 实体转义**
   - 如果代码中包含 HTML 标签，会自动转义
   - 无需手动处理

2. **性能考虑**
   - 大量代码块可能影响页面加载
   - 建议将长代码控制在合理长度

3. **浏览器兼容性**
   - 需要支持 `backdrop-filter` 的现代浏览器
   - 如果不支持，会自动降级为普通背景

## 🔧 故障排除

### 问题：代码不显示
- 检查短代码文件名是否正确（`code.html`）
- 确保 Hugo 版本支持短代码

### 问题：复制功能无效
- 检查浏览器是否支持 `navigator.clipboard`
- 确保页面在 HTTPS 环境下（某些浏览器要求）

### 问题：颜色显示不正确
- 检查 CSS 变量是否被覆盖
- 查看浏览器开发者工具确认样式生效

### 问题：移动端显示异常
- 检查视口设置：`<meta name="viewport" content="width=device-width, initial-scale=1">`
